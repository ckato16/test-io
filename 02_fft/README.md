# FFT Spectrum & Spectrogram Visualizer

Web-based audio visualization tool that displays frequency spectrum and spectrogram using Web Audio API.

## Usage

Open `index.html` in a web browser. Click buttons to record audio and visualize:
- **Spectrum**: Frequency domain visualization (1 second)
- **Spectrogram**: Time-frequency visualization (10 seconds)

Uses browser Web Audio API - no server required.
