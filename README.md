# Audio Processing and Phonics Training Projects

This repository contains multiple projects for audio processing, music analysis, and phonics pronunciation training.

## Projects

- **01_hello** - Basic audio hello world
- **02_fft** - FFT visualization
- **03_pitch** - Pitch detection
- **04_chord** - Chord detection
- **05_midi** - MIDI processing
- **06_spleeter** - Audio source separation using Spleeter
- **07_basic_pitch** - Pitch transcription using Spotify's Basic Pitch
- **08_magenta** - Music generation using Google Magenta
- **09_acid** - Audio processing
- **10_phoneme** - Phoneme pronunciation training
- **11_phonics_frontend** - Phonics frontend interface
- **12_phonics_backend** - Phonics backend with pronunciation analysis
