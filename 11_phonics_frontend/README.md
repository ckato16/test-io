# Phonics Frontend

Frontend interface for phonics pronunciation training. Displays phonics sounds organized by level (Basic/Advanced), category, and sound patterns.

## Usage

Open `index.html` in a web browser. Features:
- Select accent (British, American, Australian, etc.)
- Browse phonics sounds by category
- Listen to reference pronunciations using browser TTS
- Practice with example words

Uses browser Speech Synthesis API - no server required.
